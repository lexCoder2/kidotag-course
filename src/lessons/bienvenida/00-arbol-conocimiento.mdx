# Árbol de Conocimiento del Curso

Este árbol muestra **todos los conceptos del curso** y las relaciones de prerequisito entre ellos: qué necesitas entender primero para que el siguiente concepto tenga sentido. Úsalo como mapa de orientación en cualquier momento del curso.

<Callout variante="tip" titulo="Cómo leer este árbol">
  Las flechas significan **"es necesario para entender"**. Empieza desde la base
  (JavaScript) y sube. Los nodos en **gris claro** son conocimientos que se
  asumen como prerequisito externo al curso.
</Callout>

<Callout variante="info" titulo="Concepto clave: el ciclo req → res">
  En Express cada petición HTTP entra como objeto `req` y debe salir con una
  respuesta `res`. Los middlewares son funciones que procesan ese par antes de
  llegar al controlador final. Si llamas `next()` pasas al siguiente; si llamas
  `res.json()` terminas el ciclo.
</Callout>

<Callout variante="info" titulo="¿Por qué Socket.IO y no WebSocket puro?">
  Socket.IO añade **reconexión automática**, **rooms**, **namespaces**,
  **fallback a polling HTTP** y un protocolo de eventos nombrados. Para
  kidotag10 es ideal porque los tutores pueden reconectarse sin perder eventos y
  las aulas pueden estar en rooms separadas.
</Callout>

## Resumen de conceptos adicionales recomendados

Estos conceptos complementan el curso y se recomiendan para profundizar:

| Área      | Concepto                                                       | Por qué importa                                 |
| --------- | -------------------------------------------------------------- | ----------------------------------------------- |

| JS        | <WikiLink term="event-loop">Event loop & microtasks</WikiLink> | Depurar código async complejo                   |

| TS        | Generics `<T>`                                                 | Funciones reutilizables con tipos seguros       |
| TS        | Utility types (`Partial`, `Pick`, `Omit`)                      | Transformar tipos existentes en modelos         |

| HTTP      | HTTPS & TLS                                                    | Cifrado en tránsito, obligatorio en producción  |
| Seguridad | OWASP Top 10                                                   | Los 10 riesgos de seguridad más comunes en APIs |
| BD        | Transactions en MongoDB                                        | Operaciones atómicas multi-documento            |
| BD        | Aggregation pipeline                                           | Estadísticas y reportes complejos               |
