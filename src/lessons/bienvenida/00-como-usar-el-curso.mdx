# Cómo usar este curso

Este curso es 100% interactivo. No necesitas instalar nada para aprender — todo corre en tu navegador.

## Herramientas disponibles en cada lección

### Editor de código en vivo (Sandpack)

En muchas lecciones encontrarás un editor donde puedes **modificar el código y ver el resultado al instante**:

### Quizzes de comprensión

Al final de cada lección hay preguntas para consolidar lo aprendido. Tu progreso se guarda automáticamente.

## Cómo guardar tu progreso

Al completar cada lección, haz clic en el botón **"Marcar como completada"** que aparece al final. Tu avance se guarda en el navegador y puedes ver el porcentaje de completado en la barra lateral.

<Callout variante="tip" titulo="Consejo">
  Si tienes experiencia previa, puedes saltar libremente entre lecciones usando
  la barra lateral. Cada lección es autocontenida y puedes hacer el curso en
  cualquier orden.
</Callout>
