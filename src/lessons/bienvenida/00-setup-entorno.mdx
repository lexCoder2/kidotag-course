# Configuración del entorno

Para desarrollar y ejecutar kidotag10 necesitas instalar algunas herramientas. Aquí está todo lo que necesitas.

## Herramientas requeridas

### 1. Node.js 18+ y npm

```bash
# Verificar versión actual
node --version   # debe ser 18.x o superior
npm --version    # debe ser 9.x o superior

# Si no tienes Node.js, descárgalo de:
# https://nodejs.org (elige la versión LTS)
```

### 2. MongoDB

**Opción A — MongoDB local (para desarrollo):**

```bash
# Instalar via Homebrew (macOS)
brew tap mongodb/brew
brew install mongodb-community

# Iniciar el servicio
brew services start mongodb-community

# O en Windows, descargar el instalador de:
# https://www.mongodb.com/try/download/community
```

**Opción B — MongoDB Atlas (nube, recomendado):**

1. Crear cuenta gratuita en [mongodb.com/atlas](https://www.mongodb.com/atlas)
2. Crear un cluster (M0 Free Tier)
3. Copiar la Connection String al archivo `.env`

## Clonar y configurar kidotag10

```bash
# 1. Clonar el repositorio
git clone <url-del-repo> kidotag10
cd kidotag10

# 2. Configurar la API
cd api
npm install
cp .env.example .env
# Editar .env con tus valores

# 3. Configurar el frontend web
cd ../web
npm install
cp .env.example .env
# Editar VITE_API_URL con la URL de tu API
```

## El archivo .env de la API

```bash
# kidotag10/api/.env
PORT=3001
MONGO_URI=mongodb://localhost:27017/kidotag
JWT_SECRET=una_clave_secreta_muy_larga_y_aleatoria_aqui
FRONTEND_URL=http://localhost:5173
NODE_ENV=development
```

<Callout variante="warning" titulo="Nunca subas .env a Git">
  El archivo `.env` contiene credenciales sensibles. Está en el `.gitignore` del
  proyecto. Usa `.env.example` (sin valores reales) para documentar las
  variables requeridas.
</Callout>

## Verificar que todo funciona

```bash
# Terminal 1: iniciar MongoDB (si es local)
mongod --dbpath ./data/db

# Terminal 2: iniciar la API
cd api && npm run dev
# Debería ver: "Servidor corriendo en puerto 3001"
# Abrir: http://localhost:3001/api-docs

# Terminal 3: iniciar el frontend
cd web && npm run dev
# Abrir: http://localhost:5173
```
