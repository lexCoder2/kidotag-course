Un **Schema de Mongoose** define la estructura de los documentos en una colección: qué campos existen, qué tipos tienen, y qué validaciones aplican.

## Tipos de datos en Mongoose

```javascript
const mongoose = require("mongoose");
const { Schema } = mongoose;

const alumnoSchema = new Schema({
  nombre: { type: String }, // texto
  edad: { type: Number }, // número
  activo: { type: Boolean }, // booleano
  nacimiento: { type: Date }, // fecha
  tags: [String], // array de strings
  grupo: { type: Schema.Types.ObjectId, ref: "Grupo" }, // referencia
  contacto: {
    // subdocumento
    telefono: String,
    email: String,
  },
  metadatos: { type: Map, of: String }, // mapa clave-valor
});
```

## timestamps: createdAt y updatedAt gratis

```javascript
// Con timestamps: true en el Schema
const alumno = await Alumno.create({
  nombre: "Ana",
  matricula: "2024-001",
  grado: "3A",
});

console.log(alumno.createdAt); // 2024-01-15T08:30:00.000Z — automático
console.log(alumno.updatedAt); // 2024-01-15T08:30:00.000Z — se actualiza en cada save()
```
