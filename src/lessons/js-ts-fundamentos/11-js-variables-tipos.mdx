# Variables y tipos en JS

Si estas empezando, piensa esto: una **variable** es una cajita para guardar un dato.

- `const` = cajita que no quieres cambiar.
- `let` = cajita que puede cambiar.

## Ejemplo sencillo

```javascript
const escuela = "Primaria Juarez";
let alumnos = 230;
let hayClases = true;

alumnos = alumnos + 1;
```

<Callout variante="tip" titulo="Regla facil para iniciar">
  Usa `const` por defecto. Solo usa `let` cuando de verdad necesites cambiar el
  valor.
</Callout>

## Tipos mas comunes

- `string`: texto
- `number`: numeros
- `boolean`: verdadero/falso
- `null`: vacio intencional
- `undefined`: aun no tiene valor
