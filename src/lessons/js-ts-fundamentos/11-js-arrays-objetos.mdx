# Arrays y objetos en lenguaje humano

## Array

Un **array** es una lista:

```javascript
const alumnos = ["Ana", "Luis", "Mia"];
```

## Objeto

Un **objeto** guarda datos con etiqueta:

```javascript
const alumno = {
  nombre: "Ana",
  grado: "3A",
  presente: true,
};
```
