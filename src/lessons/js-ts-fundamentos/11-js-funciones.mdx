# Funciones que si se entienden

Una funcion es una mini maquina: entra algo, hace trabajo, y devuelve un resultado.

```javascript
function saludar(nombre) {
  return `Hola, ${nombre}`;
}
```

## Por que importan?

- Reutilizas codigo.
- Evitas copiar/pegar.
- Tu codigo se vuelve mas ordenado.
