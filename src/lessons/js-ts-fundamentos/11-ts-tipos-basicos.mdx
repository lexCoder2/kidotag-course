# TypeScript basico: tipos, interfaces y errores utiles

TypeScript es JavaScript con "cinturon de seguridad".
Te avisa errores antes de ejecutar.

## Tipos basicos

```ts
let nombre: string = "Ana";
let edad: number = 10;
let activo: boolean = true;
```

## Interface

```ts
interface Alumno {
  id: string;
  nombre: string;
  presente: boolean;
}
```
