# Promesas y async/await desde cero

Cuando pides datos al servidor, la respuesta no llega instantaneamente.
Por eso usamos codigo **asincrono**.

## Idea simple

- Tu app pide datos.
- Mientras espera, puede seguir funcionando.
- Cuando llegan, actualiza la pantalla.

```javascript
async function cargar() {
  const res = await fetch("/api/alumnos");
  return res.json();
}
```
