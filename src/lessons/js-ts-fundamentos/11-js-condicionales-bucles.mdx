# Condicionales y bucles sin dolor

Programar es tomar decisiones y repetir acciones.

- **Condicional**: "si pasa esto, haz esto".
- **Bucle**: "repite esto varias veces".

## Condicional basico

```javascript
if (asistencia >= 90) {
  console.log("Excelente");
} else {
  console.log("Necesita apoyo");
}
```

## Bucle basico

```javascript
for (let i = 1; i <= 3; i++) {
  console.log("Alumno", i);
}
```
