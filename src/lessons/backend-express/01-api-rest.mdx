# ¿Qué es una API REST?

Una **API REST** (Representational State Transfer) es una interfaz que permite a dos sistemas comunicarse a través de HTTP. kidotag10 expone una API REST que consume su aplicación web y móvil.

## HTTP: el idioma de la web

```
Petición:
  POST /api/auth/login HTTP/1.1
  Host: api.kidotag.mx
  Content-Type: application/json

  {"email": "profe@escuela.mx", "password": "Demo123!"}

Respuesta:
  HTTP/1.1 200 OK
  Content-Type: application/json

  {"ok": true, "token": "eyJ...", "usuario": {...}}
```

## Verbos HTTP y qué hacen

| Verbo    | Acción              | Ejemplo en kidotag10      |
| -------- | ------------------- | ------------------------- |
| `GET`    | Leer datos          | `GET /api/alumnos`        |
| `POST`   | Crear nuevo         | `POST /api/alumnos`       |
| `PUT`    | Reemplazar completo | `PUT /api/alumnos/:id`    |
| `PATCH`  | Actualizar parcial  | `PATCH /api/alumnos/:id`  |
| `DELETE` | Eliminar            | `DELETE /api/alumnos/:id` |

## Códigos de estado HTTP

```
2xx — Éxito
  200 OK           → Lectura/actualización exitosa
  201 Created      → Recurso creado (POST exitoso)
  204 No Content   → Eliminación exitosa (sin cuerpo)

4xx — Error del cliente
  400 Bad Request  → Body inválido o falta un campo
  401 Unauthorized → Sin autenticación (no hay token)
  403 Forbidden    → Autenticado pero sin permisos
  404 Not Found    → El recurso no existe
  409 Conflict     → Ya existe (ej: matrícula duplicada)

5xx — Error del servidor
  500 Internal Server Error → Bug inesperado en el servidor
```

// POST /alumnos

## Desafío: construye la API real con Express

El simulador de arriba muestra cómo funciona una API REST. Ahora impleméntala **tú mismo** en Express con las 4 rutas fundamentales de kidotag10:

    '/index.js': `const express = require("express");
const app = express();
app.use(express.json());

let alumnos = [
{ id: 1, nombre: "Ana García", grado: "3A", asistencia: 95 },
{ id: 2, nombre: "Luis Méndez", grado: "3A", asistencia: 78 },
{ id: 3, nombre: "Mía Torres", grado: "4B", asistencia: 100 },
];
let siguienteId = 4;

app.get("/alumnos", (req, res) => res.json({ ok: true, data: alumnos }));
app.get("/alumnos/:id", (req, res) => {
const a = alumnos.find(a => a.id === Number(req.params.id));
if (!a) return res.status(404).json({ ok: false, error: "No encontrado" });
res.json({ ok: true, data: a });
});
app.post("/alumnos", (req, res) => {
if (!req.body.nombre) return res.status(400).json({ ok: false, error: "nombre requerido" });
const nuevo = { id: siguienteId++, ...req.body };
alumnos.push(nuevo);
res.status(201).json({ ok: true, data: nuevo });
});
app.delete("/alumnos/:id", (req, res) => {
const idx = alumnos.findIndex(a => a.id === Number(req.params.id));
if (idx === -1) return res.status(404).json({ ok: false, error: "No encontrado" });
const [eliminado] = alumnos.splice(idx, 1);
res.json({ ok: true, eliminado });
});

module.exports = app;`,
}}
/>
